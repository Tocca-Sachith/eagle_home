### 💡 技術的な詳細

#### 翻訳の追加方法：

新しいテキストを翻訳対応にする場合：

1. `src/locales/en.json` と `src/locales/si.json` にキーを追加
   ```json
   {
     "mySection": {
       "title": "My Title",
       "description": "My Description"
     }
   }
   ```

2. コンポーネントで使用：
   ```tsx
   import { useLanguage } from '@/contexts/LanguageContext';
   
   function MyComponent() {
     const { t } = useLanguage();
     
     return (
       <div>
         <h1>{t('mySection.title')}</h1>
         <p>{t('mySection.description')}</p>
       </div>
     );
   }
   ```

#### ネストされたキーの使用：
```tsx
// common.home → "Home" (EN) または "මුල් පිටුව" (SI)
t('common.home')

// home.hero.title → "Building Your Dreams..." (EN)
t('home.hero.title')
```
